Klondike Solitaire

This Solitaire implementation is a WebAssembly (WASM) experiment built using Go. It leverages a custom JS-interop library I've been developing to bridge the gap between Go and the Browser DOM.

The Twist: The game logic and UI integration were developed in collaboration with Gemini, exploring how AI can assist in building functional, interactive applications on top of experimental Go-WASM stacks.

About this Project

The Grid is a conceptual playground for various software experiments. Starting from a Game of Life foundation, it's evolving into a space to test ideas around grid-based logic, cellular automata, and modern web technologies like WASM.
